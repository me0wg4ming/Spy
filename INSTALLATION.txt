================================================================================
SPY - SuperWoW Edition - SCHNELLE INSTALLATION
================================================================================

🚀 IN 3 SCHRITTEN:
================================================================================

1. ALTES SPY ENTFERNEN/UMBENENNEN
   
   Dein WoW-Ordner\Interface\AddOns\Spy → Spy_OLD


2. NEUES SPY ENTPACKEN
   
   Entpacke Spy-SuperWoW.zip nach:
   Dein WoW-Ordner\Interface\AddOns\
   
   Die Struktur sollte sein:
   Interface/AddOns/Spy/Spy.lua
   Interface/AddOns/Spy/SpySuperWoW.lua
   Interface/AddOns/Spy/Spy.toc
   ... (alle anderen Dateien)


3. WOW STARTEN
   
   Beim Einloggen siehst du:
   
   MIT SuperWoW:
   → [SpySW] SuperWoW DETECTED
   
   OHNE SuperWoW:
   → [SpySW] SuperWoW NOT DETECTED - using CombatLog fallback

================================================================================
BEFEHLE:
================================================================================

/spy                 → Öffnet Spy (wie immer)
/spyswstatus        → Zeigt SuperWoW-Status & Statistiken
/spyswdebug         → Aktiviert/Deaktiviert Debug-Modus

================================================================================
WIE FUNKTIONIERT ES?
================================================================================

MIT SuperWoW:
✅ Findet Feinde BEVOR sie angreifen
✅ Erkennt Stealth-Spieler
✅ Echte Level (kein Raten mehr)
✅ Bessere Performance

OHNE SuperWoW:
→ Spy funktioniert wie das Original (CombatLog-Methode)

Beide Modi funktionieren perfekt! SuperWoW ist optional.

================================================================================
PROBLEME?
================================================================================

Problem: Spy lädt nicht
→ Überprüfe Ordnerstruktur (Interface/AddOns/Spy/Spy.lua muss existieren)
→ Nur EINE Spy-Version installieren

Problem: Fehler beim Laden
→ Eingeben: /spyswdebug (zeigt Details)
→ Screenshot machen und melden

Problem: SuperWoW nicht erkannt
→ SuperWoW installiert? Version 1.12.1+ nötig
→ Eingeben: /spyswstatus (zeigt Status)
